Previously coded on **Mathcad**, **MATLAB**, **Scilab**, **JPHP (DevelNext)**, **Assembler**, **PureBasic**, **Delphi (7)**, **PHP (DevelStudio)**, **SFX-Macros (RU)**, **Pascal (Turbo Pascal & PascalABC.NET)** and **LogoWorlds (RU)**.
